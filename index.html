Learning the Basics of HTML

HTML (HyperText Markup Language) is the standard language for creating webpages. Here are some key elements:

1. HTML Structure

<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is my first webpage.</p>
</body>
</html>

Explanation: This is the basic structure of an HTML page.

2. Headings

<h1>Main Heading</h1>
<h2>Subheading</h2>

3. Links

<a href="https://example.com">Visit Example</a>

4. Images

<img src="image.jpg" alt="Description">
